Always use Tavily MCP tools (tavily_search, tavily_extract) for web searches instead of WebFetch or WebSearch.

**Why:** User explicitly prefers Tavily as the primary web search tool — it's configured with their API key and provides better search results.

**How to apply:** Whenever you need to search the web, look up documentation, or research anything online, reach for Tavily first. Only fall back to WebFetch/WebSearch if Tavily is unavailable or the MCP server isn't loaded.
